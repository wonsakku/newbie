## @ControllerAdvice와 @ExceptionHanlder가 어떻게 사용되는지 알아보면 좋을 것 같습니다.

<br>

> @ControllerAdvice는 Controller의 처리가 완료된 후 동작하도록 하는 애노테이션입니다.(아마도..)  
> @ExceptionHandler는 설정한 Exception에 대해 Exception을 일괄적으로 처리할 수 있게 해주는 애노테이션입니다.(아마도..)
